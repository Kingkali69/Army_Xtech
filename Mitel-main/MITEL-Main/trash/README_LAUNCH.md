# OMNI - How to Launch

## 🚀 IT'S RUNNING RIGHT NOW

The web console is **already running** and should have **automatically opened in your browser**.

If it didn't open automatically, go to:

**http://localhost:8080**

Or find your local IP and use:

**http://YOUR_IP:8080**

---

## Launch Methods (All Platforms)

### Linux
```bash
./START.sh
```
**OR** right-click `START.sh` → Execute

### Windows
**Double-click:** `START.bat`  
**OR** right-click `START.bat` → Execute  
**OR** right-click `START.ps1` → Run with PowerShell

### Android
```bash
python3 omni_web_console.py
```

---

## What Happens

1. **Server starts** on port 8080
2. **Browser opens automatically** to the console
3. **Console loads** - you see everything immediately
4. **No typing URLs** - it just works

---

## If Browser Doesn't Open

The server is still running. Just open your browser and go to:

**http://localhost:8080**

---

## Stop the Server

Press `Ctrl+C` in the terminal where it's running.

---
