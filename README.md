## 🔧 Launcher Troubleshooting

### Console Shows Nothing in Browser
**Problem:** Browser shows blank page but launcher says "Running"

**Solution:** Try direct links instead of main page:
- **http://localhost:8888** (Main console)
- **http://localhost:8888/omni_original** (OMNI infrastructure)
- **http://localhost:8888/mitel** (M.I.T.E.L. security)

**Or clear browser cache:** Ctrl+Shift+Delete → Clear cache

### High CPU Usage
**Problem:** Fan spins, system slows down

**Solution:** Use CPU-optimized launcher:
```bash
./START_SLOW_DEMO.sh  # Ultra slow for consumer hardware
```

### Process Won't Start
**Problem:** Launcher says "❌ STOPPED"

**Solution:** Check for errors:
```bash
cat console.log  # Check error logs
./OFFLINE_TEST.sh  # Run diagnostic test
```

### Port Already in Use
**Problem:** "Port 8888 already in use"

**Solution:** Kill existing processes:
```bash
./SUPER_KILL.sh  # Kill all OMNI processes
./START_SLOW_DEMO.sh  # Start fresh
```

### Windows Crashes in Admin Mode
**Problem:** System crashes when running as administrator

**Solution:** Use normal mode with CPU optimization:
- Copy files to Windows machine
- Run `launch_omni_windows.bat` normally (not as admin)
- USB quarantine will be limited but console works
