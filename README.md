# Army Xtech

Military-grade infrastructure and AI integration system.

## Features
- NEXUS AI chat interface
- OMNI orchestration engine  
- M.I.T.E.L. zero-trust security
- GhostLang P2P tunnels
- Cross-platform tool sharing

## Quick Start
1. Clone this repository
2. Install dependencies: `pip install -r requirements.txt`
3. Start NEXUS: `python3 simple_nexus_chat.py`
4. Start OMNI: `python3 launch_omni.py`
